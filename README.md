## Debugging local RNMK module

- checkout [RNMK Demo] and [RNMK][gh]
- link your local RNMK module, and launch the packager
```sh
cd rnmk-demo
npm link <local-rnmk-path>
npm install
npm start
```
- and then run the rnmk-demo xcode project
